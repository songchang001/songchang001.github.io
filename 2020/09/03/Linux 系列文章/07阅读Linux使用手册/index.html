手册中的不同区域

​ 手册分不同区域，这些区域的名字使用大写和粗体表示，且靠左对齐

NAME：名称

​ 手册页对应的名字或函数，后接简单的描述

SYNOPSIS：概要、摘要

​ 包含使用此命令的所有方法

解析：

​ 例：mkdir [OPTION]… DIRECTORY…

​ 省略号：可以有多个此类内容，即可以有多个选项参数，多个目录名

​ 粗体文字：表述要原封不动的输入，如：mkdir

​ 下划线的文字：表述要用实际内容替换，如：OPTION、DIRECTORY

​

​ 例：cp [OPTION]… [-T] SOURCE DEST ：拷贝源文件到目标文件

​ -SOURCE：源文件

​ -DEST：目录名

​ 例：cp [OPTION]… SOURCE… DIRECTORY ：拷贝一个或多个源文件到目录

​ 例：cp [OPTION]… -t DIRECTORY SOURCE… ：拷贝一个或多个源文件到目录

其他语法

​ [-hvc]：-h,-v和-c选项都是可选的，非强制性的

​ a|b：或，可以输入a或b选项，不能同时输入a和b

DESCRIPTION：描述，命令的更深入描述

​ 这个区域也会包括所有参数及用法

AUTHOR：命令的作者

COPYRIGHT：版权，用了什么许可证

SEE ALSO：扩展阅读，与此命令相关的其他命令

查找命令

apropos 命令

​ apropos是”关于…”的意思

​ apropos命令用于根据手册中的关键字来找到命令

​ 例：apropos sound ：查找如何用终端命令控制音量
